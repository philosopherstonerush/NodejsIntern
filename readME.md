# Description

API endpoints are

```

    GET - /api/blog-stats/
    
    GET - /api/blog-search?query=<your-query>

```

# Curl Commands

1) /api/blog-stats/

```

    curl http://localhost:3000/api/blog-stats/

```

2) /api/blog-search?query

```

    curl http://localhost:3000/api/blog-search?query=privacy

```
